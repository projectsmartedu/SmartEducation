# Slide 5: Inter-module Communication & Data Flow
- Diagram: User → Frontend → Backend API → ML Service (Python) → Backend → Frontend
- Backend calls ML service via HTTP/child_process
- ML service returns JSON (risk, revision, mind map)
- Example code (backend calling Python ML):
```js
const { spawn } = require('child_process');

  return new Promise((resolve) => {
```
